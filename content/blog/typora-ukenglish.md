This guide shows you how to manually add a UK spelling as spell-checker language for the Markdown editor **Typora**. The reason for this is that no matter what I tried it seemed impossible in the current version to switch from US to UK spelling, it just defaults on US spelling, and I need it to write my thesis.

First download the language file you want, in my case I go to [this](https://wordlist.aspell.net/dicts/) page and download `British, *-ize* (alternative version, Oxford spelling)`

Then decompress, change the names to `gl.aff`, `gl.dic` and drop in `C:\Users\$$your-user$$\AppData\Roaming\Typora\typora-dictionaries`. Open Typora and on the lower right switch the spelling language to **Galego - GL**. You can choose other language if you want, but the files would need to be named accordingly.

---

As extra step, if you want to set it permanently, go to `File > Preferences > Editor > Select spell check language > Galego`
